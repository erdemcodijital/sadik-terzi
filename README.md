# Sadık Terzi - Profesyonel Terzilik Hizmetleri

Bahçelievler Siyavuşpaşa'da hizmet veren Sadık Terzi için modern, kurumsal landing page.

## Özellikler

- ✨ Modern ve şık tasarım
- 📱 Tamamen responsive (mobil, tablet, desktop)
- 🎨 Özel tipografi (Cormorant Garamond + Inter)
- 🗺️ Google Maps entegrasyonu
- 📞 Direkt arama ve yol tarifi linkleri
- ⚡ Hızlı yükleme
- 🎯 SEO optimize

## Bölümler

1. **Hero** - Etkileyici giriş bölümü
2. **Hakkımızda** - İşletme tanıtımı
3. **Hizmetler** - Detaylı hizmet kartları
4. **Özellikler** - Neden tercih edilmeli
5. **Çalışma Saatleri** - Haftalık program
6. **İletişim** - Adres, telefon ve harita

## Vercel'e Deploy

### Adım 1: GitHub'a Yükleyin
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/KULLANICI_ADINIZ/sadik-terzi.git
git push -u origin main
```

### Adım 2: Vercel'de Deploy Edin
1. [vercel.com](https://vercel.com) adresine gidin
2. "New Project" tıklayın
3. GitHub repository'nizi seçin
4. Deploy'a tıklayın

**Deployment otomatik olarak tamamlanacaktır!**

## Yerel Test
